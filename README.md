## About
This project is a Web Development III final activity for the Analysis and Systems Development (ADS) degree at IFPR. It consists of a comprehensive web-based restaurant management and food delivery system, developed to demonstrate proficiency in enterprise Java technologies.

The system follows the MVC (Model-View-Controller) architecture and implements a full workflow for food ordering, from customer registration to multi-stage delivery tracking.

### Technical Requirements
The project was built to satisfy the following academic criteria:
* Backend: Java utilizing JSP/JSTL and Expression Language.
* Persistence: Robust data management using the Hibernate framework.
* Scope Management: Proper use of Session, Request, Page, and Application scopes.
* Frontend: Responsive and polished UI built with Bootstrap.
* Operations: At least 5 CRUD operations and 2 non-CRUD functional features.

### Key Features
* Complete Order Lifecycle: Tracks orders through several statuses: Under Analysis, In Preparation, Ready for Pickup, Out for Delivery, and Finished.
* Dual-Interface System:
  *  **Customer Side:** Register, login, browse menu items (dishes and drinks), and monitor real-time order progress.
  *  **Admin/Staff Side:** Manage menu catalog, validate incoming orders with justification for cancellations, and generate periodic sales reports.
* Security: Secure authentication implemented using JWT (JSON Web Token).
* Advanced Logic: Beyond standard CRUDs, the system features complex sales reporting and a dynamic order status validation engine.

The project was evaluated by Professor Valter Estevam on November 25, 2025, receiving the highest possible grade for its technical execution and design quality.
